Guest Stories
- As a guest, I want to register and log in to my account so that I can access the platform and book properties.
- As a guest, I want to search for available properties and book a stay so that I can plan my trips efficiently.
Host Stories
- As a host, I want to list my property with details and images so that I can book as soon as possible
Payment Stories
- As a guest, I want to make secure payments for my bookings so that I can complete my reservations safely.
Admin Stories
- As an admin, I want to manage users, properties, and bookings so that I can maintain the platform’s integrity and operations.
